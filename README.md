# 👊 Punching Automation

> Automate your insurance policy punching with AI!

## ✨ Features
- 📄 Upload PDF policy documents
- 🤖 AI extracts all key information automatically
- ✏️ Edit extracted data before saving
- 📊 Save records directly to Google Sheets
- ⚡ Fast and accurate

## 🚀 How to Use
1. Upload your insurance policy PDF
2. Select the associate
3. Enter commission percentages
4. Select payment status
5. Hit **Submit** — AI will extract all data
6. Review and edit if needed
7. Hit **Punch** — data saved to Google Sheets!
